## 🇧🇷 Português

**HS VenToys** é um projeto open-source onde pretendo disponibilizar uma ferramenta de criação de temas pro programa [Ventoy](https://www.ventoy.net), e talvez, no futuro adicione alguns guias de como utilizar o mesmo.

### ✨ Funcionalidades

- 🎨 Preview ao vivo da tela do GRUB
- 🖱️ Arraste e posicione elementos (menu, barra de progresso, rótulos)
- 🖼️ Upload de background, fontes (.pf2) e ícones
- 📦 Exportação em ZIP com estrutura correta (`ventoy/theme/...`)
- 💾 Salve e carregue projetos localmente no navegador
- 🌐 Totalmente estático: funciona offline e no GitHub Pages

### 🚀 Como usar

#### Online
Acesse: https://hensamuel.github.io/hs-ventoys/

#### Localmente
```bash
git clone https://github.com/hensamuel/hs-ventoys.git
cd hs-ventoys
python -m http.server
```

### 🛠 Tecnologias

- [Alpine.js](https://alpinejs.dev/) — Reatividade leve
- [Tailwind CSS](https://tailwindcss.com/) — Estilização
- [interact.js](https://interactjs.io/) — Drag & drop
- [JSZip](https://stuk.github.io/jszip/) — Geração de ZIP
- [FileSaver.js](https://github.com/eligrey/FileSaver.js) — Download de arquivos

## 🇺🇸 English 

**HS VenToys** is an open-source project where I plan to provide a theme creation tool for [Ventoy](https://www.ventoy.net). In the future, I may also add some guides on how to use it.

### ✨ Features

- 🎨 Live GRUB screen preview
- 🖱️ Drag and drop elements (menu, progress bar, labels)
- 🖼️ Upload background, fonts (.pf2) and icons
- 📦 ZIP export with correct structure (`ventoy/theme/...`)
- 💾 Save and load projects locally in the browser
- 🌐 Fully static: works offline and on GitHub Pages

### 🚀 How to use

#### Online
Visit: https://hensamuel.github.io/hs-ventoys/

#### Locally
```bash
git clone https://github.com/hensamuel/hs-ventoys.git
cd hs-ventoys
python -m http.server
```

### 🛠 Technologies

- [Alpine.js](https://alpinejs.dev/) — Lightweight reactivity
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [interact.js](https://interactjs.io/) — Drag & drop
- [JSZip](https://stuk.github.io/jszip/) — ZIP generation
- [FileSaver.js](https://github.com/eligrey/FileSaver.js) — File download
