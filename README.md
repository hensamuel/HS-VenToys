## 🇧🇷 Português

**HS VenToys** é um projeto open-source onde disponibilizo uma ferramenta visual para criação de temas do [Ventoy](https://www.ventoy.net), sem precisar editar arquivos de configuração manualmente. No futuro, pretendo adicionar também guias de utilização do Ventoy e da própria ferramenta.

### ✨ Funcionalidades

- 🎨 Preview ao vivo da tela do GRUB
- 🖱️ Arraste e posicione elementos (menu, barra de progresso, rótulos)
- 🖼️ Upload de background, fontes (.pf2) e ícones
- 📦 Exportação em ZIP com estrutura correta (`ventoy/theme/...`)
- 💾 Salve e carregue projetos localmente no navegador
- 🌐 Totalmente estático: funciona offline e no GitHub Pages

### 🚀 Como usar

#### Online
Acesse: `https://seu-usuario.github.io/hs-ventoys/`

#### Localmente
```bash
git clone https://github.com/seu-usuario/hs-ventoys.git
cd hs-ventoys
# Abra o index.html no navegador ou use um servidor local:
python -m http.server
# Acesse http://localhost:8000
